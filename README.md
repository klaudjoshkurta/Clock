### Clock

A lightweight, reliable alarm clock for Android with recurring alarms, snooze, tones, and a clean UI.

### Features
- Create, edit, enable/disable multiple alarms
- One-time or repeating schedules (daily / weekdays / custom days)
- Snooze support (configurable duration)
- Alarm sound selection + optional vibration
- Dark mode
- Privacy-friendly (no login)

### Tech Stack (suggested)
- Kotlin
- Jetpack Compose (UI)
- Room (local storage)
- WorkManager + AlarmManager (scheduling)
- Hilt (DI)
- Kotlin Coroutines + Flow (state)
